```bash
strace -o logs -s1024 -fe execve man bash
```
- `-s` - длина выводимой строки
- `-f` - отслеживание дочерних процессов
- `-e` - какой системный вызов отслеживается
- `-o` - куда сохранить трассировку
В файл будут сохранены все системные вызовы [[execve]], выполненные во время выполнения программы.
Пример:
```bash
execve("/usr/local/bin/nroff", ["nroff", "-mandoc", "-rLL=175n", "-rLT=175n", "-Tutf8"], 0x5644e41148d0 /* 36 vars */ <unfinished ...>
```
