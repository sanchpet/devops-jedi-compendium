## Development

1. Clone the repo
2. Create [venv](https://docs.python.org/3/library/venv.html)

    ```shell
    python3 -m venv .venv
    source .venv/bin/activate
    ```

3. Install requirements

    ```shell
    pip install -r requirements.txt
    ```

4. Enable [pre-commit](https://pre-commit.com/#install)

    ```shell
    pre-commit install
    ```
